Edit Profile/Account Details

  • Edit Profile and Account Details

    1. Select the "Edit" button on the bottom right hand corner of the profile page.


    2. Edit the profile as you wish.

      If you wish to change the password or email, click "Edit Email/Password" button.


    3. Re-enter your password.


    4. Edit the details as you wish.

      Deleting the account also can be done through this page.



  • Edit Profile Picture

    1. Bring the mouse over the profile picture area, and click the "Add new profile picture button".


    2. Click the browse button, browse for an image, select the image and click the "Upload" button.


    3. After uploading, the profile picture can be edited (the same way as adding) or deleted.
