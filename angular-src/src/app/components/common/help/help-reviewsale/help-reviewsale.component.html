Add a Review on an Item/Service

  • Add/Edit a Review

    1. Select the item by searching for the item or through your profile page (if you have bought, fevorited or requested the item).

      Scroll to the bottom and click the "Add Review" button.


    2. Type the review and select the rating you want to give for the item and submit.


    3. Click the "Edit" button to edit the review and "Delete" button to delete the review (if you have added a review).



The procedure will be identical when reviewing a Service.
